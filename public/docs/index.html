Introduction

Employee API docs

Base URL: http://employee-api.test

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

To authenticate requests, include an Authorization header with the value "Bearer {YOUR_AUTH_KEY}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

To obtain a token use the Login endpoint (email: test@example.com, password: password)

Auth

Login

Example request:
curl --request POST \
    "http://employee-api.test/api/v1/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"test@example.com\",
    \"password\": \"password\"
}"

const url = new URL(
    "http://employee-api.test/api/v1/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "test@example.com",
    "password": "password"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Index Employee

Example request:
curl --request GET \
    --get "http://employee-api.test/api/v1/employees?include=employeeAddress&filter%5Bfull_name%5D=sint&filter%5Bemail%5D=adipisci&filter%5Bposition%5D=mollitia&sort=qui&page%5Bnumber%5D=15&page%5Bsize%5D=20" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://employee-api.test/api/v1/employees"
);

const params = {
    "include": "employeeAddress",
    "filter[full_name]": "sint",
    "filter[email]": "adipisci",
    "filter[position]": "mollitia",
    "sort": "qui",
    "page[number]": "15",
    "page[size]": "20",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());
